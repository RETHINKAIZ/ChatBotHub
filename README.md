# ChatBotHub - 多模型聊天聚合 Chrome 插件

一个强大的 Chrome 浏览器插件，将主流 AI 大模型集成到统一界面，支持浮动应用栏模式。

## 功能特性

- 🤖 **自定义模型** - 支持添加任意 AI 模型，上传图标、设置名称和链接
- 📁 **分组管理** - 支持创建自定义分组，将模型按类别整理
- 🎨 **浮动应用栏** - 开启后在页面右侧显示圆形图标栏，鼠标悬停显示名称
- 📌 **拖拽排序** - 支持拖拽模型图标调整顺序
- 🔍 **快速搜索** - 支持搜索模型名称和描述
- 💾 **配置导入导出** - 支持导出/导入配置，方便迁移和备份
- 📋 **右键菜单** - 支持右键选中文本后快速打开 AI 模型

## 安装方法

### 开发模式安装

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 开启右上角的 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择 `ChatBotHub` 文件夹
6. 插件安装完成！

### 固定到工具栏（推荐）

1. 点击浏览器右上角的拼图图标 🧩
2. 找到 "ChatBotHub"
3. 点击图钉图标固定到工具栏

## 使用方法

### 弹出窗口

1. 点击浏览器工具栏中的插件图标 🤖
2. 在弹出窗口中可以看到所有支持的模型（按分组展示）
3. 点击任意模型卡片，在新标签页中打开

### 浮动应用栏（新功能）

1. 点击插件图标，点击右上角的 ⚙️ 设置按钮
2. 开启 **侧边栏应用栏** 开关
3. 在任意网页右侧会显示圆形模型图标栏
4. 点击图标即可在新标签页打开对应模型
5. 鼠标悬停在图标上可显示模型名称
6. 支持拖拽图标调整顺序
7. 点击底部的 − 按钮可收起应用栏

### 添加自定义模型

1. 点击插件图标，点击 **添加模型** 按钮
2. 填写模型名称、系统链接、简单描述
3. 选择分组（或创建新分组）
4. 上传图标（可选，支持图片文件）
5. 选择颜色（用于默认图标背景）
6. 点击保存

### 管理分组

1. 点击 **管理分组** 按钮
2. 查看现有分组列表
3. 添加新分组：输入名称和 emoji 图标
4. 删除分组：点击分组右侧的删除按钮（默认分组不可删除）

### 搜索模型

1. 在搜索框中输入关键词
2. 实时过滤模型名称和描述
3. 按 `⌘K` (Mac) 或 `Ctrl+K` (Windows) 快速聚焦搜索框

### 导入导出配置

1. 点击设置按钮 ⚙️
2. 点击 **导出配置** 下载配置文件
3. 点击 **导入配置** 选择之前导出的文件

## 注意事项

### 账户登录

- 首次使用各模型时，需要在打开的页面中登录账户
- 登录状态会保存在浏览器中，下次自动登录
- 如需切换账号，请在对应模型官网退出后重新登录

### 图标上传

- 支持 PNG、JPG、SVG 等常见图片格式
- 建议上传正方形图片（如 64x64 或 128x128）
- 文件大小不超过 1MB
- 如果不上传图标，将使用模型名称首字母和选定颜色作为默认图标

## 技术说明

- **Manifest V3** - 使用最新的 Chrome 插件规范
- **Storage API** - 本地存储用户模型、分组和偏好设置
- **Content Scripts** - 注入页面实现浮动应用栏
- **Context Menus API** - 右键菜单集成

## 开发

### 调试

1. 在 `chrome://extensions/` 中找到插件
2. 点击 "检查视图 - Service Worker" 查看后台日志
3. 右键点击插件图标，选择 "检查弹出内容" 查看 popup 日志
4. 在任意网页按 F12 打开开发者工具，查看 Console 中 content script 的日志
