# 模型图标说明

此目录用于存放各 AI 模型的图标文件。由于版权原因，请自行下载或使用以下替代方案：

## 临时方案

在正式图标添加之前，插件会使用 CSS 生成的字母图标作为占位符。

## 获取图标的方法

1. **官方来源**：访问各模型官网下载官方 favicon
2. **使用工具**：使用 https://www.google.com/s2/favicons?domain=chat.openai.com&sz=64 类似的 API
3. **手动下载**：从各官网下载品牌资源

## 推荐图标尺寸

- 48x48 PNG（推荐）
- 64x64 PNG
- 32x32 PNG

## 临时解决方案

当前代码已使用 CSS 生成彩色圆形图标作为占位符，即使没有真实图标文件，插件也能正常显示和使用。
